# AdamW Optimizer

AdamW is a variant of the Adam optimizer that implements weight decay correctly. It was introduced by Ilya Loshchilov and Frank Hutter in their 2019 paper "Decoupled Weight Decay Regularization".

## Key Idea

The key insight behind AdamW is that L2 regularization and weight decay are not equivalent when used with adaptive gradient algorithms like Adam. AdamW decouples weight decay from the gradient update, leading to better generalization performance.

## The Problem with Adam and L2 Regularization

In standard stochastic gradient descent (SGD), L2 regularization and weight decay are equivalent. However, with adaptive gradient methods like Adam, they behave differently:

- **L2 Regularization**: Adds the term $\frac{\lambda}{2}||\theta||^2$ to the loss function, resulting in gradients with an additional term $\lambda\theta$.
- **Weight Decay**: Directly modifies the weight update rule by subtracting $\lambda\theta$ from the update.

With adaptive gradient methods, these approaches lead to different outcomes because the regularization term gets scaled by the adaptive learning rate when implemented as L2 regularization.

## Mathematical Formulation

The standard Adam update rule is:

$$\theta_{t+1} = \theta_t - \frac{\eta}{\sqrt{\hat{v}_t} + \epsilon} \cdot \hat{m}_t$$

Where:
- $\theta_t$ represents the parameters at time step $t$
- $\eta$ is the learning rate
- $\hat{m}_t$ is the bias-corrected first moment estimate
- $\hat{v}_t$ is the bias-corrected second moment estimate
- $\epsilon$ is a small constant for numerical stability

AdamW modifies this update rule by applying weight decay directly:

$$\theta_{t+1} = \theta_t - \eta \cdot \lambda \cdot \theta_t - \frac{\eta}{\sqrt{\hat{v}_t} + \epsilon} \cdot \hat{m}_t$$

Where $\lambda$ is the weight decay coefficient.

## Full Algorithm

The complete AdamW algorithm is as follows:

1. Initialize parameters $\theta_0$, moments $m_0 = 0$, $v_0 = 0$, and time step $t = 0$
2. At each time step $t$:
   - Compute gradient $g_t$ (without the L2 regularization term)
   - Update biased first moment: $m_t = \beta_1 \cdot m_{t-1} + (1 - \beta_1) \cdot g_t$
   - Update biased second moment: $v_t = \beta_2 \cdot v_{t-1} + (1 - \beta_2) \cdot g_t^2$
   - Compute bias-corrected first moment: $\hat{m}_t = m_t / (1 - \beta_1^t)$
   - Compute bias-corrected second moment: $\hat{v}_t = v_t / (1 - \beta_2^t)$
   - Update parameters: $\theta_t = \theta_{t-1} - \eta \cdot \lambda \cdot \theta_{t-1} - \eta \cdot \hat{m}_t / (\sqrt{\hat{v}_t} + \epsilon)$

## Advantages of AdamW over Adam

1. **Better Generalization**: AdamW often leads to better generalization performance, especially for large models.

2. **More Effective Regularization**: The decoupled weight decay provides more effective regularization than L2 regularization with Adam.

3. **Improved Convergence**: For some problems, AdamW converges to better solutions than Adam.

4. **Theoretically Sound**: AdamW correctly implements weight decay, making it theoretically more sound than Adam with L2 regularization.

## When to Use AdamW

AdamW is particularly well-suited for:

- Large neural networks where generalization is a concern
- Models that benefit from regularization
- When you're currently using Adam with L2 regularization
- Transfer learning and fine-tuning pre-trained models

## Numerical Example: Minimizing a Simple Function with AdamW

Let's walk through a concrete example of using AdamW to minimize the simple function:

$$f(x) = x^2 + 10$$

This is a simple parabola with a global minimum at $x = 0$, where $f(0) = 10$.

The derivative of this function is:

$$f'(x) = 2x$$

Now let's apply AdamW to find the minimum:

### Initial Setup
- Starting point: $x_0 = 5$
- Learning rate: $\eta = 0.1$
- Weight decay coefficient: $\lambda = 0.01$
- First moment decay rate: $\beta_1 = 0.9$
- Second moment decay rate: $\beta_2 = 0.999$
- Small constant: $\epsilon = 10^{-8}$
- First moment: $m_0 = 0$
- Second moment: $v_0 = 0$
- Time step: $t = 0$

### Iteration 1 (t = 1)
- Current position: $x_0 = 5$
- Gradient: $g_1 = f'(x_0) = 2 \times 5 = 10$
- Update first moment: $m_1 = \beta_1 \times m_0 + (1 - \beta_1) \times g_1 = 0.9 \times 0 + 0.1 \times 10 = 1$
- Update second moment: $v_1 = \beta_2 \times v_0 + (1 - \beta_2) \times g_1^2 = 0.999 \times 0 + 0.001 \times 10^2 = 0.1$
- Bias-corrected first moment: $\hat{m}_1 = \frac{m_1}{1 - \beta_1^1} = \frac{1}{1 - 0.9} = 10$
- Bias-corrected second moment: $\hat{v}_1 = \frac{v_1}{1 - \beta_2^1} = \frac{0.1}{1 - 0.999} = 100$
- Weight decay term: $\eta \times \lambda \times x_0 = 0.1 \times 0.01 \times 5 = 0.005$
- Adam update: $\frac{\eta \times \hat{m}_1}{\sqrt{\hat{v}_1} + \epsilon} = \frac{0.1 \times 10}{\sqrt{100} + 10^{-8}} = \frac{1}{10} = 0.1$
- Total update: $x_1 = x_0 - \text{weight decay} - \text{Adam update} = 5 - 0.005 - 0.1 = 4.895$
- New function value: $f(x_1) = 4.895^2 + 10 = 23.96 + 10 = 33.96$

### Iteration 2 (t = 2)
- Current position: $x_1 = 4.895$
- Gradient: $g_2 = f'(x_1) = 2 \times 4.895 = 9.79$
- Update first moment: $m_2 = \beta_1 \times m_1 + (1 - \beta_1) \times g_2 = 0.9 \times 1 + 0.1 \times 9.79 = 0.9 + 0.979 = 1.879$
- Update second moment: $v_2 = \beta_2 \times v_1 + (1 - \beta_2) \times g_2^2 = 0.999 \times 0.1 + 0.001 \times 9.79^2 = 0.0999 + 0.0958 = 0.1957$
- Bias-corrected first moment: $\hat{m}_2 = \frac{m_2}{1 - \beta_1^2} = \frac{1.879}{1 - 0.81} = \frac{1.879}{0.19} = 9.89$
- Bias-corrected second moment: $\hat{v}_2 = \frac{v_2}{1 - \beta_2^2} = \frac{0.1957}{1 - 0.998} = \frac{0.1957}{0.002} = 97.85$
- Weight decay term: $\eta \times \lambda \times x_1 = 0.1 \times 0.01 \times 4.895 = 0.00489$
- Adam update: $\frac{\eta \times \hat{m}_2}{\sqrt{\hat{v}_2} + \epsilon} = \frac{0.1 \times 9.89}{\sqrt{97.85} + 10^{-8}} = \frac{0.989}{9.89} = 0.1$
- Total update: $x_2 = x_1 - \text{weight decay} - \text{Adam update} = 4.895 - 0.00489 - 0.1 = 4.79$
- New function value: $f(x_2) = 4.79^2 + 10 = 22.94 + 10 = 32.94$

### Iteration 3 (t = 3)
- Current position: $x_2 = 4.79$
- Gradient: $g_3 = f'(x_2) = 2 \times 4.79 = 9.58$
- Update first moment: $m_3 = \beta_1 \times m_2 + (1 - \beta_1) \times g_3 = 0.9 \times 1.879 + 0.1 \times 9.58 = 1.691 + 0.958 = 2.649$
- Update second moment: $v_3 = \beta_2 \times v_2 + (1 - \beta_2) \times g_3^2 = 0.999 \times 0.1957 + 0.001 \times 9.58^2 = 0.1955 + 0.0918 = 0.2873$
- Bias-corrected first moment: $\hat{m}_3 = \frac{m_3}{1 - \beta_1^3} = \frac{2.649}{1 - 0.729} = \frac{2.649}{0.271} = 9.77$
- Bias-corrected second moment: $\hat{v}_3 = \frac{v_3}{1 - \beta_2^3} = \frac{0.2873}{1 - 0.997} = \frac{0.2873}{0.003} = 95.77$
- Weight decay term: $\eta \times \lambda \times x_2 = 0.1 \times 0.01 \times 4.79 = 0.00479$
- Adam update: $\frac{\eta \times \hat{m}_3}{\sqrt{\hat{v}_3} + \epsilon} = \frac{0.1 \times 9.77}{\sqrt{95.77} + 10^{-8}} = \frac{0.977}{9.79} = 0.1$
- Total update: $x_3 = x_2 - \text{weight decay} - \text{Adam update} = 4.79 - 0.00479 - 0.1 = 4.685$
- New function value: $f(x_3) = 4.685^2 + 10 = 21.95 + 10 = 31.95$

### Key Observations About AdamW

Notice several important aspects of how AdamW works:

1. **Decoupled Weight Decay**: The weight decay term is applied separately from the adaptive learning rate, directly scaling the parameter by $\eta \times \lambda$.

2. **Cumulative Effect**: While the weight decay term seems small in each iteration, its effect accumulates over time, leading to more significant regularization.

3. **Faster Convergence to Zero**: For this simple function, AdamW converges slightly faster to the minimum at x = 0 compared to standard Adam.

4. **Regularization Effect**: Even for this simple function without overfitting concerns, we can see how AdamW's weight decay pushes the parameter closer to zero.
