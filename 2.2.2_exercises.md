## Multi-Index DataFrames

### Exercise 5: Create a Multi-Index DataFrame
1.  Create a DataFrame with a multi-level index and multi-level columns. Use the following data structure:
    ```python
    data = {
        ('Q1', 'Sales'): {('USA', 'East'): 100, ('USA', 'West'): 120, ('EU', 'North'): 80},
        ('Q1', 'Profit'): {('USA', 'East'): 20, ('USA', 'West'): 25, ('EU', 'North'): 15},
        ('Q2', 'Sales'): {('USA', 'East'): 110, ('USA', 'West'): 130, ('EU', 'North'): 90},
        ('Q2', 'Profit'): {('USA', 'East'): 22, ('USA', 'West'): 28, ('EU', 'North'): 18}
    }
    ```
2.  Create a DataFrame from this dictionary and print it.
