Instructions for the desktop app project:
1. Press windows key and type *powershell* and open it.
2. Copy and paste following after `>`:
```bash
conda activate ds
```
5. Press Enter.
>**Note:** Or choose one of your existing environments and activate it. Anything other than `base` is fine.
6. Copy and paste following after `>`:
```bash
conda install ttkbootstrap
```
7. Press Enter.
>**Note:** Python comes with tkinter library. So there is no need to install it. `ttkbootstrap` is a library that provides a modern and customizable theme for tkinter.
8. Create a new folder in your workspace and name it `simple_desktop_app`.
10. Inside the `simple_desktop_app` folder, create a new file and name it `miles_to_km.py`. Pay attention to the file extension.
11. Start writing the code for the app.
12. Save the file.
13. Navigate to the `simple_desktop_app` folder in the anaconda powershell terminal by typing `cd C:\Users\your_username\Desktop\div\simple_desktop_app`. Make sure to replace `your_username` with your actual username.
14. Run the app by typing `python miles_to_km.py`.
